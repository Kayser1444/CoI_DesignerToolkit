# CoI Designer Toolkit
Tools and quality-of-life improvements for Captain of Industry blueprint designers.

## Design principle: designer-only, consumer-free

This mod is intended for **blueprint creators**. Players who download and use your blueprints do **not** need to have this mod installed. Every feature must produce standard, vanilla-compatible blueprint data — the mod only assists with the act of creating or editing blueprints, never with consuming them.

## Features

### Update blueprint
Select a blueprint in your blueprint book and click **Update** to replace its contents with a fresh area selection. The blueprint's name, description, overlap settings, and position in the folder are all preserved.
